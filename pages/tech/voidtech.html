Anchorfling Void, Hypersurge.

Overview

Void tech is two anchored blocks (build & module) far away connected to a constraint (motors for example). It requires a fling module to separate them. This was originally used to bypass the void barrier (~-5000) to be able to fly below it without dying. Void tech automatically gives axis lock. For more info, read the axis lock void page. Though void tech doesnt just give immunity to the FallenPartsDestroyHeight limit (~-5000 void barrier). For example, if the rootpart is moving, it messes up interpolation and causes surge and invinciblity, and sometimes fling when using other constraints.

Surge Surge is the most known effect of void tech. It allows the player to "go crazy" serversidely, looks like he gets flung on other people's side but is normal clientsidely. The surge can be different depending on the fling and if it flung well. Examples : Microsurge, hypersurge, invisibility, etc...
